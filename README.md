# Number Guessing Beast

Welcome to **Number Guessing Python Script**, the ultimate ultra-fast number predictor.

## Features
- Lightning-fast number guessing for any range (0 to billions)
- Undo (`U`) and Redo (`R`) support for precise control
- Done (`D`) clears attempts and ends the session
- - Optimized for maximum speed and efficiency
- Fully compatible with Python and Termux

## How to Run

Step 1: Install Python if you haven't

```pkg install python -y```

Step 2: Clone the repository

```git clone https://github.com/GTK-HUB0/NumberGuessing.py.git```

Step 3: Navigate to the folder

```cd NumberGuessing.py```

Step 4: Run the script

```python number_guesser.py```

## Commands During Game
- `H` : Your number is higher than the guess
- `L` : Your number is lower than the guess
- `D` : Done, clears attempts and ends session
- `U` : Undo last guess
- `R` : Redo last undone guess
